Migration Status

Migration Required. The following schema changes were applied via npx prisma db push on the local development database:
— 2 new enums added: LeaveType (SICK, CASUAL, EARNED, MATERNITY, PATERNITY, UNPAID, OTHER) and LeaveStatus (PENDING, APPROVED, REJECTED, CANCELLED)
— New table leave_requests with columns: id, userId, type, startDate (@db.Date), endDate (@db.Date), totalDays, reason, status, approvedById, approvalNote, approvedAt, createdAt, updatedAt
— 3 indexes: on userId, on status, on (startDate, endDate)
— 2 new FK relations added to users table: leaveRequests (UserLeaveRequests) and leaveRequestApprovals (LeaveRequestApprovals)

Production deployment: npx prisma migrate deploy will run automatically on Render deploy. All changes are fully additive — existing tables and rows are unaffected.
